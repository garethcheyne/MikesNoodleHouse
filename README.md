# Mikes Noodle House.

This project is split into two application, front end and back end.

## Setup Process.
1. run py .\server\manage.py
2. run py .\server\sampledata.py
3. Start Flask Server Back End
4. Start Tk Front End.

## Start Flask Back.
exe run_backend.bat 

_and leave running.


## Start Tk Front End.
exe run_client.bat
